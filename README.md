# 🍳 CookSmart  
### Smart Cooking & Pantry Management System  

---

## 📖 Project Description  

CookSmart is a full-stack Smart Cooking & Pantry Management System designed to help users manage their pantry, discover recipes, track nutrition, and reduce food wastage.  

The system allows users to store ingredients digitally, receive recipe recommendations based on available pantry items, calculate nutritional values automatically, and interact socially by sharing and reviewing recipes.  

CookSmart integrates smart recommendation, diet filtering, and grocery tracking features to create a complete modern cooking assistant platform.
